# Advent of Code 2024 - Clojure

A Clojure project for solving Advent of Code 2024 puzzles.

## Usage

To start a REPL:
```bash
clj -M:nrepl
```

To run tests:
```bash
clj -X:test
```

## Project Structure

- `src/` - Source code
- `test/` - Test files
- `resources/` - Resource files and puzzle inputs
